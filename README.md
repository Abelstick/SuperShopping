## 📖 ¿Qué es SuperShopping?

SuperShopping es una aplicación web para gestionar el **inventario del hogar** y organizar las compras de forma más inteligente. La idea es tener control de lo que hay en casa, lo que falta y **cuánto se va a gastar antes de salir a comprar**.

No busca ser solo una lista de compras, sino una **herramienta práctica** para tomar mejores decisiones al momento de gastar.

---

## ✨ Funcionalidades

| Función | Descripción |
|---|---|
| 📦 **Inventario** | Registra y controla productos del hogar |
| 🗂️ **Categorías** | Organiza productos por categorías |
| 🛍️ **Modo Compra** | Seguimiento en tiempo real en el supermercado |
| 💰 **Presupuesto** | Define cuánto gastar antes de salir |
| 📊 **Dashboard** | Visualiza tu información de compras |
| 👥 **Colaborativo** | Comparte con otros usuarios del hogar |
| 🌙 **Modo oscuro** | Interfaz adaptable día/noche |

---

## 🛍️ Modo Compra

Incluye un modo pensado para usarse **directamente mientras compras**:

> - Ver qué productos necesitas
> - Registrar lo que ya vas comprando
> - Mantenerte dentro del presupuesto definido
> - Tener claridad en tiempo real de **cuánto llevas gastado**

La idea es evitar compras impulsivas y no salir con sorpresas en caja.

## 🚀 Instalación y uso

```bash
# Clona el repositorio
git clone https://github.com/Abelstick/SuperShopping.git
cd SuperShopping

# Instala dependencias
npm install

# Inicia en modo desarrollo
npm run dev
```

## 🏗️ Estructura del proyecto

El proyecto está organizado por **features** para mantener el código limpio y escalable:

```
src/
├── modules/
│   ├── inventory/       # Gestión de productos del hogar
│   ├── categories/      # Organización de productos
│   ├── shopping/        # Planificación y ejecución de compras
│   ├── dashboard/       # Visualización de datos
│   ├── budget/          # Control previo a la compra
│   └── settings/        # Preferencias del usuario
├── hooks/               # Lógica desacoplada de la UI
└── components/          # Componentes reutilizables
```
